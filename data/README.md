# Cached data

## `retraction_watch_dois.csv`

A two-column cache (`OriginalPaperDOI`, `RetractionDOI`) of the public
Retraction Watch retraction database. Refreshed daily by
[`.github/workflows/update-retraction-watch.yml`](../.github/workflows/update-retraction-watch.yml).

**Source and credit.** The Retraction Watch Database is provided by
[Retraction Watch](https://retractionwatch.com/) / The Center for Scientific
Integrity, made openly available in partnership with Crossref under the
[CC-BY 4.0](https://creativecommons.org/licenses/by/4.0/) license. The full
dataset and column documentation live at
<https://gitlab.com/crossref/retraction-watch-data>.

If you build on this data, please credit **Retraction Watch** explicitly.
